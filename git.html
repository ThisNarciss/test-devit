=== GIT ===

Варіант 1

Виконую команду git clone git@example.com:example/test.git в терміналі, клоную репо до свого локального сховища

↓

Копіюю всі файли свого робочого коду та переношу в клонований репо

↓

Виконую команду git add . в терміналі, додаю всі файли в індекс

↓

Роблю комміт командою git commit -m "some commit text" в терміналі, додаю всі файли в індекс

↓

І остання команда git push в терміналі, надсилаю зміни до віддаленого репо

Варіант 2

Переходжу до каталогу де знаходиться мій робочий код командою cd "dirname" в терміналі

↓

Ініціалізую локальний репо командою git init в терміналі

↓

Виконую команду git add . в терміналі, додаю всі файли в індекс

↓

Роблю комміт командою git commit -m "some commit text" в терміналі, додаю всі файли в індекс

↓

Зв'язую локальний репо з віддаленим командою git remote add origin git@example.com:example/test.git в терміналі

↓

І остання команда git push --set-upstream origin master в терміналі, надсилаю зміни до віддаленого репо в гілку master
